¿Como funciona?

Este es un proceso para la obtencion de una beca exclusiva, que corresponde a una suscripción de un mes en nuestra plataforma. Funciona de la siguiente manera: Son 2 cursos que tendrás que completar al 100%, una vez completados se habilitará la opción para ingresar tus datos para una solicitud de beca. ¿Te consideras lo suficientmente bueno para aceptar el reto?. Recuerde que el formulario debe ser rellenado por un padre o apoderado mayor de edad.
